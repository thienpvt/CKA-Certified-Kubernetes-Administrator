# Blueprint Bravo — CKA Mock Exam #2

A 2-hour, 17-question timed mock exam with a completely different question draw
from Blueprint Alpha. No question overlap between the two blueprints ensures
candidates get fresh practice on every attempt.

> **Disclaimer:** Not real CKA exam content; independently authored. Targets v1.35 CKA blueprint.

## Estimated Time Budget

The sum of per-question `estimatedMinutes` for this blueprint falls in the range
**120–145 minutes**, which exceeds the 120-minute exam duration. This is expected
because the zero-overlap constraint with Blueprint Alpha forces inclusion of
heavier questions. The exam timer still runs a **2:00:00 countdown** — time
pressure is part of the simulation.

## Quick Start

```bash
cka-sim exam blueprint-bravo
```
