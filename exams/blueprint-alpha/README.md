# Blueprint Alpha — CKA Mock Exam

A 2-hour, 17-question mock exam targeting the Kubernetes v1.35 CKA blueprint.

## Quick Start

```bash
cka-sim exam blueprint-alpha
```

## Blueprint Details

- **Duration:** 2 hours (120-minute countdown)
- **Questions:** 17, drawn from all 5 domain packs
- **Weighting:** Storage 10% · Workloads & Scheduling 15% · Services & Networking 20% · Cluster Architecture 25% · Troubleshooting 30%
- **Pass mark:** 66/100

## Estimated Time Budget

The sum of per-question `estimatedMinutes` for this blueprint is **129 minutes**
(range: 120–130). This exceeds the original MOCK-01 requirement of 110–120 minutes
because the underlying domain-pack questions were authored at 6–11 minutes each for
thorough drill practice. The exam timer still runs a **2:00:00 countdown** — most
candidates will finish with time to spare on questions they know well.

This deviation is documented here and flagged for ROADMAP amendment in Phase 8.

## Question Order

Questions are interleaved across domains to simulate the real exam's
context-switching demands. The order is fixed for blueprint-alpha — use
`blueprint-bravo` (Phase 8) for a different draw without repetition.
