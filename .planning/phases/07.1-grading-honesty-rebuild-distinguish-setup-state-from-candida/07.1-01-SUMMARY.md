## What Was Built

1. **cka-sim/lib/baseline.sh** — Two exported functions:
   - `cka_sim::baseline::capture <ns>`: Two-query approach (namespaced + cluster-scoped filtered by question-ID prefix), atomic write, kind normalization via `ascii_downcase`, exits non-zero on kubectl failure.
   - `cka_sim::baseline::is_candidate_modified <kind> <name> [-n <ns>]`: Generation-first/rv-fallback logic, back-compat returns 0 when baseline unavailable.

2. **3 new grade helpers in lib/grade.sh**:
   - `assert_changed_since_setup`: Passes iff resource modified since baseline (via is_candidate_modified).
   - `assert_generation_delta_ge`: Passes iff generation delta >= threshold N.
   - `assert_resource_candidate_authored`: Passes iff resource NOT in baseline AND currently exists.

3. **Catalog ownership schema** — `ownership:` optional field (values: candidate-required, setup-allowed) added to lint-traps.sh validation. 5 text-based content-bug traps tagged `setup-allowed`.
