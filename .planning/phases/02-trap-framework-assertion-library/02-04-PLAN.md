Case-file invocation pattern (per PATTERNS.md "tests/cases/*.sh"):
```bash
#!/bin/bash
# tests/cases/traps_default-sa-used.sh — verifies detect_default_sa_used.
set -uo pipefail
: "${CKA_SIM_ROOT:?must be set by run.sh}"
: "${CKA_SIM_TEST_FIXTURES_DIR:?must be set by run.sh}"

source "$CKA_SIM_ROOT/lib/traps.sh"
source "$CKA_SIM_ROOT/tests/lib/assert.sh"

# ---------- hit ----------
export CKA_SIM_TEST_CURRENT="default-sa-used/hit"
result=$(cka_sim::trap::detect_default_sa_used cka-sim-test webapp || true)
expect_eq "$result" "default-sa-used" "hit fires trap"

# ---------- miss ----------
export CKA_SIM_TEST_CURRENT="default-sa-used/miss"
result=$(cka_sim::trap::detect_default_sa_used cka-sim-test webapp || true)
expect_empty "$result" "miss does not fire trap"

# ---------- benign ----------
export CKA_SIM_TEST_CURRENT="default-sa-used/benign"
result=$(cka_sim::trap::detect_default_sa_used cka-sim-test other-pod || true)
expect_empty "$result" "benign shape does not fire trap"
```

Pass/fail aggregation: case files inherit run.sh's `set -uo pipefail` (no -e). Each
expect_* returns 0 or 1; the case overall returns 0 only if every expect_* returned 0.
Use a local accumulator pattern at the bottom of each case file:
```bash
case_failed=0
expect_eq ... || case_failed=1
expect_empty ... || case_failed=1
exit "$case_failed"
```
