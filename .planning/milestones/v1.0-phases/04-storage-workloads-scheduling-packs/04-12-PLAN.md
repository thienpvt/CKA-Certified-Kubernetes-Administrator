    # workloads-scheduling/03-configmap-secret-env-volume

    **Domain:** Workloads & Scheduling  |  **Estimated time:** 8 minutes

    A ConfigMap `q03-app-config` and Secret `q03-app-secret` exist in `${CKA_SIM_LAB_NS}`. Create a Pod that consumes both.

    ## Tasks

    1. Create a Pod `q03-app` in `${CKA_SIM_LAB_NS}` running image `busybox:1.36` with a long-lived command (e.g. `sleep 3600`).
    2. Expose the ConfigMap key `APP_MODE` as an environment variable named `APP_MODE` using `valueFrom.configMapKeyRef`.
    3. Mount the Secret `q03-app-secret` as a **read-only** volume at `/etc/app-secrets`; the key `API_KEY` must be readable at `/etc/app-secrets/api-key`.

    ## Constraints

    - Use `valueFrom` for env (do NOT copy the literal value into Pod spec).
    - Secret volume must be read-only.

    ## Verify yourself

    ```
    kubectl exec -n ${CKA_SIM_LAB_NS} q03-app -- printenv APP_MODE           # production
    kubectl exec -n ${CKA_SIM_LAB_NS} q03-app -- cat /etc/app-secrets/api-key
    ```

    #!/bin/bash
    set -uo pipefail
    : "${CKA_SIM_LAB_NS:?}"
    : "${CKA_SIM_ROOT:?}"
    source "$CKA_SIM_ROOT/lib/grade.sh"
    source "$CKA_SIM_ROOT/lib/traps.sh"

    kubectl wait --for=condition=Ready pod/q03-app -n "$CKA_SIM_LAB_NS" --timeout=60s 2>/dev/null || true

    cka_sim::grade::assert_pod_ready "$CKA_SIM_LAB_NS" q03-app
    cka_sim::grade::assert_field_eq pod q03-app \
      '{.spec.containers[0].env[?(@.name=="APP_MODE")].valueFrom.configMapKeyRef.name}' \
      'q03-app-config' \
      -n "$CKA_SIM_LAB_NS"
    cka_sim::grade::assert_field_eq pod q03-app \
      '{.spec.containers[0].env[?(@.name=="APP_MODE")].valueFrom.configMapKeyRef.key}' \
      'APP_MODE' \
      -n "$CKA_SIM_LAB_NS"

    # Find secret volume mount readOnly=true
    cka_sim::grade::assert_field_eq pod q03-app \
      '{.spec.containers[0].volumeMounts[?(@.mountPath=="/etc/app-secrets")].readOnly}' \

      CKA_SIM_GRADE_PASSED=$(( CKA_SIM_GRADE_PASSED + 1 ))
