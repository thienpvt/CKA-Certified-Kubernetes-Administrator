- **Impact:** high — creates a live Pod that auto-restarts on reboot, occupies worker resources, breaks D-09.
- **Asset:** live 1+2 cluster's kube-system ns and worker resource pool.
- **Mitigation:** Pass G forbidden-command guard rejects `/etc/kubernetes/` writes and copies. Acceptance criterion greps for `/etc/kubernetes/` absence in all 4 scripts. Grader uses ONLY `kubectl apply --dry-run=client` (client-side, no API submission).
- **Verification:** `bash cka-sim/scripts/lint-packs.sh` pass G green. P09 VERIFICATION.md manual check: `ls /etc/kubernetes/manifests/` before/after drill returns identical listing.

### T-6-17: python3 yaml.safe_load doesn't catch the tab-indentation error as expected
- **Threat:** YAML parsers are forgiving; a duplicate-key break might parse silently on some PyYAML versions, so the `static-pod-manifest-bad-yaml` trap never fires.
- **Impact:** medium — trap assertion becomes a no-op; question loses a named failure mode.
