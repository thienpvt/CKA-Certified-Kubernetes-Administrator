# Plan 08-02: Superseded-Content Banners

## Objective

Add a markdown note block at the top of three legacy README files pointing users to `cka-sim/`. No existing content below the banner is modified.

## Tasks

### Task 1: Add banners to three files

Prepend the following block (6 lines including blank separator) to the TOP of each file, before any existing content:

```markdown
> **Note:** The content below is superseded by the interactive exam simulator.
> See [`cka-sim/`](./cka-sim/) for trap-aware drills, timed mocks, and automated grading.
> This content remains for reference but is no longer actively maintained.

```

Files:
1. `README.md` (root) — relative link: `./cka-sim/`
2. `exercises/README.md` — relative link: `../cka-sim/`
3. `mock-exams/README.md` — relative link: `../cka-sim/`

Rules:
- Insert at line 1 (before any existing `#` heading)
- Blank line after the block quote separates it from existing content
- Do NOT modify, reorder, or delete any existing content below

## Verification

- Each file starts with `> **Note:**`
- Existing content (first heading, body) is unchanged below the banner
- Relative links resolve correctly from each file's location
