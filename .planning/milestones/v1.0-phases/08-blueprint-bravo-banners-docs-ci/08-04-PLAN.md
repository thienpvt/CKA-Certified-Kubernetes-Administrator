# Plan 08-04: CI Wiring

## Objective

Create `validate-local.sh` (yamllint + shellcheck for all cka-sim files) and extend the GitHub Actions workflow with a shellcheck job.

## Tasks

### Task 1: Create `cka-sim/scripts/validate-local.sh`

New bash script. Structure:

```bash
#!/bin/bash
set -euo pipefail

CKA_SIM_ROOT="$(cd "$(dirname "$0")/.." && pwd)"
source "$CKA_SIM_ROOT/lib/colors.sh"
source "$CKA_SIM_ROOT/lib/log.sh"

header "validate-local"

errors=0

# Pass 1: yamllint on all YAML files
info "yamllint: cka-sim/**/*.yaml"
find "$CKA_SIM_ROOT" -name '*.yaml' -o -name '*.yml' | while read -r f; do
  yamllint -d '{extends: default, rules: {line-length: {max: 200}, truthy: disable, document-start: disable, comments-indentation: disable, indentation: {indent-sequences: whatever}}}' "$f" || errors=$((errors + 1))
done

# Pass 2: shellcheck on all .sh files
info "shellcheck: cka-sim/**/*.sh"
find "$CKA_SIM_ROOT" -name '*.sh' | while read -r f; do
  shellcheck -x -s bash "$f" || errors=$((errors + 1))
done

if [[ $errors -gt 0 ]]; then
  err "$errors file(s) failed validation"
  exit 1
fi

ok "all files pass"
```

Notes:
- `-x` flag tells shellcheck to follow `source` directives
- `-s bash` forces bash dialect
- Uses the same color/log library as other scripts
- Exit 0 only if both passes succeed
- Error counter tracks failures across both passes

### Task 2: Extend `.github/workflows/validate.yml`

Add a new job `shellcheck` after the existing `bash-tests` job:

```yaml
  shellcheck:
    name: ShellCheck + yamllint (cka-sim)
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Install tools
        run: |
          sudo apt-get update && sudo apt-get install -y shellcheck
          pip install yamllint

      - name: Run validate-local.sh
        run: bash cka-sim/scripts/validate-local.sh
```

The existing `bash-tests` job already runs `lint-packs.sh` (which enforces pack lint rules CI-03) and `lint-deprecated-strings.sh` (CI-02). This new job adds shellcheck (CI-01) and yamllint coverage for cka-sim YAML.

## Verification

- `.github/workflows/validate.yml` has three jobs: yamllint, bash-tests, shellcheck
- shellcheck job installs shellcheck + yamllint and runs validate-local.sh
