## 1. Create directory structure
Create:
- `cka-sim/bin/`
- `cka-sim/lib/cmd/`

## 2. Write `cka-sim/lib/colors.sh`
Exports `RED`, `GREEN`, `YELLOW`, `BLUE`, `BOLD`, `NC`. TTY-detect so colors turn off when stdout isn't a terminal (avoids ANSI escapes polluting logs / scripts consuming stdout).

## 3. Write `cka-sim/lib/log.sh`
Source colors.sh. Export functions:
- `info "msg"` ‚Äî prints "  $msg" to stderr
- `ok "msg"` ‚Äî green "‚úì $msg" to stderr
- `warn "msg"` ‚Äî yellow "! $msg" to stderr
- `err "msg"` ‚Äî red "‚úó $msg" to stderr
- `die "msg"` ‚Äî err + `exit 1`
- `header "title"` ‚Äî bold separator banner
- `verbose "msg"` ‚Äî prints iff `CKA_SIM_VERBOSE=1`

Every function takes its message through `printf '%s' "$*"` (no format-string injection).

## 4. Write `cka-sim/lib/preflight.sh`
Placeholder helpers that Plan 02 fills in. For Plan 01 it's an empty-but-sourceable file declaring expected function names:
```bash
# Declared here, implemented in Plan 02:
# cka_sim::preflight::check_binaries
# cka_sim::preflight::check_cluster_nodes
# cka_sim::preflight::check_kubeconfig
# cka_sim::preflight::check_ssh_batchmode
# cka_sim::preflight::check_state_dirs
```

## 5. Write `cka-sim/bin/cka-sim`
The router. Responsibilities:
- `set -euo pipefail`
- Resolve `CKA_SIM_ROOT` as `"$(cd "$(dirname "$0")/.." && pwd)"`; export it
- Source `lib/colors.sh` and `lib/log.sh`
- Dispatch first arg to `lib/cmd/<cmd>.sh` via `exec` for: `bootstrap`, `doctor`, `list`, `version`, `drill`, `exam`, `score`
- Treat `-h`, `--help`, `help`, empty ‚Üí `lib/cmd/help.sh`
- Unknown subcommands ‚Üí `die "unknown subcommand: $cmd ‚Äî run 'cka-sim --help'"`
- chmod +x

## 6. Write `cka-sim/lib/cmd/help.sh`
Print usage:
```
cka-sim ‚Äî CKA Exam Simulator

Usage: cka-sim <command> [args]

Commands:
  bootstrap   Prepare cluster for exam practice (SSH, env, state dirs)
  doctor      Check cluster readiness; exits 0 if healthy
  list        Show available packs, blueprints, history
  drill       Practice a single question (not yet implemented ‚Äî phase 3)
  exam        Take a timed mock exam (not yet implemented ‚Äî phase 7)
  score       View a past session report (not yet implemented ‚Äî phase 7)
  version     Print version + repo checksum
  help        Show this message

See cka-sim/README.md for details.
```
