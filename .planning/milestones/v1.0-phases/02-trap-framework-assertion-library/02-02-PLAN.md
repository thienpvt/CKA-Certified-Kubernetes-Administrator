    b. `cka_sim::trap::detect_missing_dns_egress <namespace> <netpol-name>`
       - Echoes `missing-dns-egress` if NetworkPolicy has egress rules (i.e. is restricting egress) AND none of those egress rules permit UDP/53.
       - kubectl call: `kubectl get networkpolicy "$np" -n "$ns" -o json 2>/dev/null`
       - Algorithm: parse with jq.
         * If `.spec.policyTypes` does NOT include `Egress`, return early (not an egress-restricting policy â†’ no trap).
         * If `.spec.egress` is missing or empty, return early (egress-deny-all is intentional in some cases â€” but our trap fires when there ARE egress rules but none allow DNS).
         * Iterate `.spec.egress[].ports[]`; if any has `protocol: UDP` (or unset) AND `port: 53`, this is a hit-miss â†’ return early.
         * Otherwise echo `missing-dns-egress`.
       - Implementation:
         ```bash
         cka_sim::trap::detect_missing_dns_egress() {
           local ns="${1:?}" np="${2:?}"
           local json
           json=$(kubectl get networkpolicy "$np" -n "$ns" -o json 2>/dev/null) || return 0
           [[ -n "$json" ]] || return 0
           local has_egress
           has_egress=$(echo "$json" | jq -r '.spec.policyTypes // [] | index("Egress") != null' 2>/dev/null)
           [[ "$has_egress" == "true" ]] || return 0
           local has_dns
           has_dns=$(echo "$json" | jq -r '
             [.spec.egress[]?.ports[]?
              | select((.protocol // "TCP") == "UDP")
              | select((.port // 0) == 53 or (.port // "") == "53")
             ] | length > 0' 2>/dev/null)
           if [[ "$has_dns" != "true" ]]; then
             echo "missing-dns-egress"
           fi
         }
         ```

    c. `cka_sim::trap::detect_hostpath_pv_without_nodeaffinity <pv-name>`

         cka_sim::trap::detect_hostpath_pv_without_nodeaffinity() {

           [[ -n "$json" ]] || return 0
           local has_hostpath has_nodeaffinity
